Typescript

TypeScript是Javascript的超集，遵循最新的ES5/ES6规范。Typescript扩展了Javascript语法。

基础类型

TS中冒号后面的都为类型标识

  • 布尔、数字、字符串类型

  • 元组类型

    限制长度个数，类型一一对应

  • 数组类型

  • 枚举类型

    可以枚举，也可以反举

  • any 类型

    放弃 ts 的类型检测

  • null 和 undefined

    任何类型的子类型，如果 strictNullChecks 的值为 true, 则不能把 null 和 undefined 赋给其他类型

  • void 类型

    只接受 null， undefined。一般用于函数的返回值

  • never 类型

    任何类型的子类型，never 代表不会出现的值。不能把其他类型赋值给 never

  • Symbol 类型

    表示独一无二

  • BigInt 类型

    与 number 类型不兼容

  • Object 对象类型

    object 表示非原始类型
