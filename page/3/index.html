  1. 设置全局 git 的账号和邮箱，并生成 ssh key
  2. 到 github 设置新电脑的 ssh key
  3. 把 etc 下的 host 添加为 github 地址 140.82.112.4 github.com ，定向解析
  4. 打开 cmd 输入 ipconfig /flushdns，清理 DNS 缓存

2020.1

10月份遇到的问题

  1. 用 wx 的 jssdk 时，wx.ready() 用 Promise 返回出去，不然调用获取地理位置的方法(getLocation)的时候可能会获取不到。

  1. select 选择框记得 options 的值绑定为字符串类型，不然默认的表单验证类型为字符串，到时候如果绑定的值为数字，将会视为内容不存在，导致校验不通过。

  2. uniapp 转微信小程序 toJSON 方法报错，那就去那个 .vue 文件内的 methods 里定义一个 toJSON 的空白方法，但这个方法会影响到微信原生的其他组件的效果，如底部导航，因为这相当于你定义了一个 toJSON 覆盖了微信原生的 toJSON 方法。
