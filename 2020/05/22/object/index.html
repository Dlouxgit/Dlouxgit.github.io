对象

只要不是五种基本数据类型(ES5 五种，ES6 加了 Symbol)，都是对象。

对象的值保存在堆内存中，对象的引用（即变量）是保存在栈内存中的。

创建对象

  • 字面量创建

    var obj = {} ，也叫直接创建

  • 构造函数创建

    var obj = new Object()

  • Object.create() 该方法使用现用对象来提供新对象的 __proto__ ，例
