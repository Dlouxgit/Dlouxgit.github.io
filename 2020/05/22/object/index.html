对象

只要不是五种基本数据类型(ES5 五种，ES6 加了 Symbol)，都是对象。

对象的值保存在堆内存中，对象的引用（即变量）是保存在栈内存中的。

垃圾回收机制

如果一个对象没有变量指向它，它就会被回收。重点是理解 等号左右两边，它们都是独立的一部分，不要把它们看成整体。 右边只是一个对象而已，左边的变量也不是对象的名字，对象没有名字，只是指向它的地址而已。

  • 构造函数创建

    var obj = new Object()
